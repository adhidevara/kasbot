01

Autentikasi

POST /api/auth/login Login dengan nomor WA ▾

Login menggunakan nomor WhatsApp. User harus sudah terdaftar via onboarding bot. Mengembalikan JWT token berlaku 7 hari.

ℹ Tidak memerlukan password. Identitas diverifikasi melalui kepemilikan nomor WhatsApp.
Request Body
FieldTipeWajibKeterangan
nomor_wastringrequiredNomor WA format internasional. Contoh: 628123456789
curl
curl -X POST https://kalaproject-prod.up.railway.app/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"nomor_wa": "628123456789"}'
json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 604800,
  "user": {
    "nomor_wa": "628123456789",
    "nama_bisnis": "Warung Pak Anto",
    "plan": "basic"
  }
}
json
{
  "success": false,
  "message": "Nomor WA tidak terdaftar"
}
GET /api/auth/wa/status Status koneksi WhatsApp ▾
🔐 Memerlukan autentikasi

Mengecek status koneksi WhatsApp bot. Gunakan endpoint ini untuk polling status sebelum menampilkan QR.

connected
WA terhubung aktif
waiting_qr
Menunggu scan QR
disconnected
Tidak terhubung
curl
curl https://kalaproject-prod.up.railway.app/api/auth/wa/status \
  -H "Authorization: Bearer <token>"
json
{
  "status": "connected"  // connected | waiting_qr | disconnected
}
GET /api/auth/wa/qr Ambil QR code untuk scan ▾
🔐 Memerlukan autentikasi
⚠ Hanya tersedia saat status waiting_qr. Jika status sudah connected, endpoint ini mengembalikan error 400.
curl
curl https://kalaproject-prod.up.railway.app/api/auth/wa/qr \
  -H "Authorization: Bearer <token>"
json
{
  "qr": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAA..."
  // Render: <img src={qr} />
}
json
{
  "success": false,
  "message": "QR tidak tersedia. Status: connected"
}

Transaksi

GET /api/transaksi List transaksi dengan filter ▾
🔐 Memerlukan autentikasi
Query Parameters
ParameterTipeDefaultKeterangan
tipestring—pemasukan atau pengeluaran
daristring—Tanggal mulai. Format: YYYY-MM-DD
sampaistring—Tanggal akhir. Format: YYYY-MM-DD
limitnumber20Maks per halaman (max 100)
pagenumber1Halaman
curl
curl "https://kalaproject-prod.up.railway.app/api/transaksi?tipe=pemasukan&dari=2026-03-01&limit=10" \
  -H "Authorization: Bearer <token>"
json
{
  "data": [
    {
      "id": "uuid-...",
      "tipe": "pemasukan",
      "total": 150000,
      "kategori": "Penjualan",
      "sumber_input": "text",
      "transaksi_at": "2026-03-08T10:30:00Z"

  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 142
  }
}
