API Reference

REST API untuk integrasi dashboard KasBot. Semua endpoint menggunakan JSON dan autentikasi JWT Bearer token.

Base URL
localhost:3000
Format
application/json
Auth
Bearer JWT
Authentication
🔑
Semua endpoint kecuali POST /api/auth/login membutuhkan header:

Authorization: Bearer <token>

Token didapat dari response login dan berlaku selama 7 hari.
POST /api/auth/login Login dengan nomor WhatsApp ▼
Request Body
FieldTypeStatusDeskripsi
nomor_wa string required Format internasional tanpa +. Contoh: 628123456789
CURL
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "nomor_wa": "628123456789"
  }'
JSON
{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "uuid-xxx",
    "nomor_wa": "628123456789",
    "nama_bisnis": "Warung Bu Sari",
    "plan": "trial"
  }
}
JSON
{
  "success": false,
  "message": "Nomor WA tidak terdaftar"
}
WhatsApp Connection
connected
WA terhubung aktif

Menunggu scan QR

GET /api/auth/wa/status Cek status koneksi WhatsApp 🔑 JWT ▼
CURL
curl http://localhost:3000/api/auth/wa/status \
  -H "Authorization: Bearer <token>"
JSON
{
  "status": "connected", // connected | waiting_qr | disconnected
  "connected_at": "2026-03-08T10:00:00Z"
}
GET /api/auth/wa/qr Ambil QR code untuk scan WhatsApp 🔑 JWT ▼
ℹ️
QR hanya tersedia saat status waiting_qr. Polling GET /wa/status setiap 3 detik setelah scan untuk deteksi koneksi berhasil.
CURL
curl http://localhost:3000/api/auth/wa/qr \
  -H "Authorization: Bearer <token>"
JSON
{
  "qr_base64": "data:image/png;base64,iVBORw0KGgo..."
  // Render: <img src={qr_base64} />
}
JSON
{
  "success": false,
  "message": "QR tidak tersedia. Status: connected"
}

Transaksi
GET /api/transaksi List transaksi dengan filter & pagination 🔑 JWT ▼
Query Params
ParamTypeStatusDeskripsi
tipestringoptionalpemasukan atau pengeluaran
daristringoptionalISO date. Contoh: 2026-03-01
sampaistringoptionalISO date. Default: hari ini
limitnumberoptionalDefault: 20. Maks: 100
pagenumberoptionalDefault: 1
CURL
curl "http://localhost:3000/api/transaksi?tipe=pengeluaran&dari=2026-03-01&limit=20&page=1" \
  -H "Authorization: Bearer <token>"
JSON
{
  "data": [
    {
      "id": "uuid-xxx",
      "total": 43000,
      "tipe": "pengeluaran",
      "sumber_input": "foto",
      "deskripsi": "ruanguang coffee...",
      "transaksi_at": "2026-03-07T19:48:00Z"
    }
  ],
  "meta": {
    "total": 48,
    "page": 1,
    "limit": 20,
    "total_pages": 3
  }
}

POST /api/transaksi Input transaksi manual via dashboard 🔑 JWT ▼
Request Body
FieldTypeStatusDeskripsi
totalnumberrequiredNominal transaksi dalam Rupiah
tipestringrequiredpemasukan atau pengeluaran
deskripsistringoptionalKeterangan transaksi
transaksi_atstringoptionalISO datetime. Default: sekarang
itemsarrayoptionalArray detail item (lihat struktur di bawah)
penyesuaianarrayoptionalDiskon / pajak / biaya tambahan
CURL — minimal
curl -X POST http://localhost:3000/api/transaksi \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "total": 50000,
    "tipe": "pemasukan",
    "deskripsi": "Jual nasi goreng 2 porsi"
  }'
CURL — dengan items & penyesuaian
curl -X POST http://localhost:3000/api/transaksi \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "total": 43000,
    "tipe": "pengeluaran",
    "deskripsi": "Beli bahan baku",
    "transaksi_at": "2026-03-08T09:00:00Z",
    "items": [
      {
        "nama": "Tepung Terigu",
        "qty": 2,
        "harga": 15000,
        "satuan": "kg"
      },
      {
        "nama": "Telur",
        "qty": 10,
        "harga": 2000,
        "satuan": "butir"
      }
    ],
    "penyesuaian": [
      {
        "nama": "Diskon Langganan",
        "nilai": 5000,
        "tipe": "potongan"
      }
    ]
  }'
JSON
{
  "success": true,
  "id": "uuid-baru",
  "message": "Transaksi berhasil disimpan"
}
JSON
{
  "success": false,
  "message": "Field 'total' dan 'tipe' wajib diisi"
}

Anomali & Insight
GET /api/anomali/insight Insight keuangan 7 hari terakhir 🔑 JWT ▼
⚠️
Hanya tersedia untuk plan basic dan pro. Plan trial akan mendapat response 403.
CURL
curl http://localhost:3000/api/anomali/insight \
  -H "Authorization: Bearer <token>"
JSON
{
  "total_pemasukan": 1200000,
  "total_pengeluaran": 850000,
  "laba_kotor": 350000,
  "gross_margin": "29.2%",
  "jumlah_transaksi": 12,
  "periode": "2026-03-01 — 2026-03-08"
}
JSON
{
  "success": false,
  "message": "Fitur ini hanya tersedia untuk plan Basic/Pro"
}
